WHITECAP MORTGAGE NEWSLETTER - GoHighLevel Email Builder Instructions
=====================================================================

The newsletter has been split into 8 separate HTML blocks because GHL
email builder truncates large single-paste HTML.

HOW TO BUILD THE EMAIL IN GHL:
------------------------------

1. Open GHL Email Builder (Marketing > Emails > Create)
2. Start with a BLANK template
3. For each block below, drag a "Code" element into the builder
4. Paste the HTML from that block file into the Code element
5. Repeat for all 8 blocks in order:

   Block 1: block1-header.html        - Header + issue date
   Block 2: block2-video.html         - Weekly video thumbnail + link
   Block 3: block3-rates.html         - Mortgage market update
   Block 4: block4-market-stats.html  - Sussex and Worcester county stats
   Block 5: block5-events.html        - Local coastal events with images
   Block 6: block6-loan-tip.html      - Loan program tip of the week
   Block 7: block7-niche-programs.html - 6 niche program cards
   Block 8: block8-cta-footer.html    - CTA button + footer

TIPS:
-----
- Set the email background color to #f0f7ff (light blue) in GHL settings
- Each block is self-contained and will not break other blocks
- To skip a section, just do not include that block
- Niche Programs uses stacked single-column layout for better GHL support

EDITING EACH WEEK:
------------------
Look for EDIT comments in each block. Those mark where you swap content.
